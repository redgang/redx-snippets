## *.vue文件代码片段用法
> 导入 VS code IDE

### 公共配置
- log
    - console
- impm
    - 导入node_modules文件
- impc
    - 导入components组件
- impvc
    - 导入view 下面的common组件
- impvx
    - 导入vuex mapGetters与mapActions

### .vue文件配置
- cvue
    - 创建含有templete，script，style引用
- v-tpl
    - 创建templete标签
- v-scr
    - 创建script标签，包含vuex，export
- v-sty
    - 引用style样式

### vuex中modules文件及常量
- cvx
    - 生成vuex2.0 modules文件
- cst
    - 新建常量，add、delete、modify、get
